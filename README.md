# 🎁 ERC-20 Token Airdrop Script

Script ini digunakan untuk mengirim token ERC-20 ke banyak wallet sekaligus secara otomatis.

---

## Cara Menjalankan

1. Install dependensi:
   ```bash
   pip install -r requirements.txt
---

2. Edit main.py:

   * Masukkan RPC URL (Infura/Alchemy)

   * Masukkan private key wallet pengirim token

   * Masukkan alamat contract ERC20 token

3. Buat file recipients.txt berisi daftar alamat penerima token (satu per baris)

4. Jalankan scriptnya sekarang:
   ```bash
   python main.py
